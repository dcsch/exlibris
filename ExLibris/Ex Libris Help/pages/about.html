About Ex Libris

Features

  • Opens ProDOS images stored in ProDOS-order and DOS-order.
  • Opens DOS images
  • Handles raw image files (.dsk, .po, .do) and 2IMG files (.2mg).
  • Opens CFFA media, handling multiple partitions.
  • Displays directory entries.
  • Displays individual files as:
    • Applesoft BASIC
    • Text
    • 6502 Disassembly
    • Hi-Res Graphics
    • Hex dump
  • Drag-and-drop files from a disk image to a destination (such as Finder)
